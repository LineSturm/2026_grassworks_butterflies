## Content of the repository

1.  **Data**: the folder `data` contains
    -   `Raw` data of butterfly abundances (.csv)
    -   `Raw` data of butterfly species' traits (.csv)
    -   `Raw` data of natural regions of Germany (.csv)
    -   `Raw` data of blurred site coordinates (.csv)
    -   `Raw` data of sites variables (.csv)
    -   `Raw` data of vegetation variables (.csv)
    -   `Raw` data of management appropriateness (.csv)
    -   `Raw` data of landscape variables (.csv)
    -   `processed` data of butterfly sampling coverage for Hill number of order 0 (.csv)
    -   `processed` data of butterfly diversity at equal sampling coverage for Hill number of order 2 (.csv)
    -   `processed` data of all combined data (.csv)
2.  **Outputs**: the folder `outputs` contains
    -   The figures generated (.png)
    -   The tables generated (.csv/.xlsx)
    -   The models calculated (.Rdata) and their summary statistics (.png/ .txt/ .xlsx)
3.  **R**: the folder `R` contains
    -   Scripts to prepare the data (.R)
    -   Scripts to explore the data (.R)
    -   Scripts to calculate all models (.R)
    -   Scripts to generate all figures (.R)
